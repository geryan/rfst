######## Misc linux commands:

To delete everything in a directory run
rm /path/to/dir/*

To remove all sub-directories and files
rm -r /path/to/dir/*


###### Spartan-specific commands

	checks data use in /data/cefphs/punim0995

	cancel job by job name
