Misc linux commands:

check_home_usage
	checks data use in /home/ryange
	
check_project_usage
	checks data use in /data/cefphs/punim0995

sinteractive --time=<dd-hh:mm:ss> --partition=<partition>
  specify partition name

scancel --jobid=<job number>
	cancel job

du -h --max-depth=<N>
  disk usage in human readable form to some max folder depth N

module spider <something>
  get list of modules with something in name e.g. R, LANDIS
